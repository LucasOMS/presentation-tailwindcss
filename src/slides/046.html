Les composants CSS ont pour objectif de réunir un ensemble de styles. Ils sont plus spécifiques que ce qui est défini dans la base mais doivent pouvoir être surchargé par les utilitaires.

Dans ma présentation j'ai souvent parlé du fait que les problemes induits par les utilitaires étaient la plupart du temps réglés par l'utilisation de framework et de composants. C'est ici une facon de les faire directement en CSS.
