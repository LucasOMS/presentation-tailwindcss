Propriété arbitraire : le CSS exact

CSS arbitraire

<div class="[display:grid]">...</div>

Utiliser des variables CSS

<div class="flex gap-(--gutter) [--gutter:2px] lg:[--gutter:4px]">...</div>
Bon, je pense qu'on a vu pas mal de chose qui ont pu vous hérisser le poil, on va aller jusqu'à l'ultime affront : faire du CSS inline dans les utilitaires.

→

En effet en utilisant tout entre crochet, cela va générer le code exact que vous avez écrit. Alors, question évidente : pourquoi faire ça ? Pourquoi vouloir à ce point que notre PR soit refusée ? Et bien l'argument majeur, c'est qu'on peut utiliser les variants sur cet utilitaire.

Ainsi, là on n'aurait pas pu le faire dans l'attribut style, il est possible de définir une règle qui répondra à des media queries, à d'autres conditions.

→

L'exemple donné par tailwind, et je pense que c'est en effet la seule utilisation qu'on devrait tolérer, c'est de pouvoir définir des variables CSS.
Ici on voit bien que la variable sera dépendante de la taille de l'écran, et j'utilise un utilitaire pour utiliser cette variable définie.
