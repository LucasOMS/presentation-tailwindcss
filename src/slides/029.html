La sémantique de Tailwind

Utilitaire de base

variant:
propriété
- valeur
!
Une fois qu'on a l'utilitaire de base → on peut le suffixer avec un !, cela aura pour effet de définir la règle CSS avec le modificateur !important.

→

Pour chaque utilitaire, on va pouvoir ajouter des variants avant l'utilitaire de base, on peut en ajouter autant que l'on souhaite. On verra juste après tout ce que cela nous permet de spécifier.

Pour chaque variant, il sera possible de préfixer par not- pour inverser le variant. En fonction de comment ces variants sont définis, cela marchera plus ou moins bien.
