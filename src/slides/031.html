Vous avez également accès à d'autres média queries assez classique, surtout prévu pour l'accéssiblité. Quelques exemples en vrac :

        →

Vous pouvez cibler le mode sombre à partir des préférences de l'utilisateur, par exemple pour mettre le texte en blanc sur fond noir. Par défaut, cela utilise forcément la préférence utilisateur, si vous voulez utiliser le mode sombre ou un autre theme sans utiliser la préférence navigateur par défaut, mais via du code, nous verrons plus tard comment faire.

        →

Vous pouvez n'utiliser certains effets que si l'utilisateur n'a pas désactivé les animations.

        →

Vous pouvez renforcer les contrastes si l'utilisateur le demande.

        →

Vous pouvez cibler l'orientation de l'écran, par exemple pour mettre le texte à gauche en portrait et centré en paysage.

        →

Vous pouvez cibler l'impression, par exemple pour mettre le texte en plus gros.

        →

Vous pouvez cibler la précision du pointer, par exemple pour rendre un bouton plus large lorsque l'utilisateur n'utilise pas une souris ou un stylet. Ici ca cible le pointeur principal, si vous voulez cibler au moins un des pointeurs disponibles, vous pouvez utiliser any-pointer comme utilitaire.



Sur tous ces variants, le not- marche très bien, vous pouvez par exemple utiliser un not-print:hidden pour avoir un élément uniquement visible à l'impression.
