Extension : Gimli Tailwdind

Il y a une extension disponible sur Chrome et Firefox qui s'appelle Gimli Tailwind, j'ai pas regardé si elle est dispo sur les autres navigateurs.

Elle permet de facilement visualiser le résultat sur un élément, elle permet également de voir quel breakpoint sont actifs actuellements.

J'émet un avis reservé dessus, surtout car elle ne prend pas en compte notre configuration de Tailwind mais se base sur les configurations par défaut. Je vous la cite car c'est la première que vous trouverez en cherchant une extension pour Tailwind.
