La configuration a changé entre la version 3 et 4, meme si la version 4 peut encore fonctionner avec la technique de la version 3.

En version 3 cela se fait par un fichier javascript tandis qu'en v4 cela passe par un fichier css.

Vous avez à l'écran une idée d'à quoi cela peut ressembler. On va rentrer dans le détails de quelques configuration. A partir d'ici, je vais commencer à alterner entre les différentes versions.
