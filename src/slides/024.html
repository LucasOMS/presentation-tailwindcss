La philosophie Tailwind

    Comme je l'ai déjà dit dans la présentation, Tailwind est un framework CSS qui ne fournit pas un ensemble fini d'utilitaires, mais qui génère tout ceux qui suivent une certaines syntaxe.

    On aura une classe utilitaire par prorprité CSS qui doit être modifiée. Si vous savez écrire le CSS, alors vous saurez écrire le tailwind de base. Pour tout ce qui est plus avancé, nous verrons comment l'écrire, et on pourra discuter de la pertinence de l'utiliser ou pas.
