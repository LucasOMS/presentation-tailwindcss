Personnaliser Tailwind

Générer les utilitaires avec !important 

// tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  // ...
  important: true,
};
/* tailwind.css */
@import "tailwindcss" important;
Comme je l'ai dit précédemment, et contrairement à Bootstrap, les utilitaires tailwind par défaut n'utilisent pas !important. C'est logique puisque le flag !important peut se définir à la volée sur un utilitaire.

→

En v3, il suffit d'utiliser la propriété important dans la configuration.

V

En v4, ca se configure au moment de l'import de tailwind dans le fichier CSS.
