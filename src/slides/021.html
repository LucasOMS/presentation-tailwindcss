Est-ce que quelqu'un a une idée du rendu, ne serait-ce que sur petit écran ?

Alors oui, c'est atroce, et je pense que Tailwind pouvait difficilement se mettre une plus grosse balle dans le pieds sur sa page d'accueil. Mais si le HTML est illisible, je voudrais quand meme préciser que le CSS équivalent n'est pas mieux. Alors quand je dis équivalent, je n'ai pas juste réécrit tous les utilitaires, j'ai vraiment écrit le CSS qui aurait ce rendu si on ajoutait des classes qui ont du sens, avec les medias queries.

Notez que dans le CSS je n'ai mis que des valeurs en dur, ca pourrait être encore plus long à lire avec des variables SCSS et/ou des variables CSS. Je suis parti de données naïves où un espacement est égal à 1em et les écrans sont des incréments de 150px à partir de 300px
