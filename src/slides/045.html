Personnaliser Tailwind

Ajouter un configuration de base

@layer base {
    color: var(--color-gray-950);
}
Tailwind v3 comme v4 utilise les layers CSS pour organiser ses styles, dans l'ordre : base, components, utililities.

Le layer base devrait contenir le style global ayant pour vocation le reset des styles par défaut du navigateur, ainsi que les styles de typographie de base.

Pour ajouter un élément au layer de base de tailwind, il suffit de le cibler en CSS.
