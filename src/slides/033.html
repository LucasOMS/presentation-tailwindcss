La sémantique de Tailwind

Variant : pseudo class

Empty

<div class="empty:hidden">...</div>

Even

<li class="even:bg-gray-300">Élément</li>

First

<li class="first:rounded-t-xl">Élément</li>

Last

<li class="last:rounded-b-xl">Élément</li>

Has checked

<label class="text-red-500 has-checked:text-green-500">
Case à cocher
<input type="checkbox">
</label>
Tailwind fourni à peu près toutes les variants possibles pour cibler les pseudo classes CSS.

→

Ici on masque un élément si il n'a pas d'enfant. Cela peut servir quand le contenu est conditionné et qu'on ne veut pas de toute l'arborescence lorsqu'elle est vide.

→

On peut cibler les éléments pairs pour leur appliquer un fond différent.

→

On peut cibler le premier et le dernier élément d'une liste pour ajouter des arrondis en haut et en bas.

→

On peut vérifier si l'élément contient un enfant coché par exemple pour styliser le label.
