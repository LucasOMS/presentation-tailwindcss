Les outils à avoir

Surcharger les utilitaires facilement

Tailwind de base

<div class="bg-red-500">Font rouge</div>
<div class="bg-blue-500!">Font bleu</div>

Tailwind merge

<!-- Impossible de surcharger le background -->
<div class="bg-blue-500!"></div>
twMerge('bg-blue-500', 'bg-red-500'); // → bg-blue-500
twMerge('px-2 py-1', 'p-3'); // → p-3
On a vu que si on n'initialise pas tailwind avec le flag important, il sera possible de surcharger n'importe quel utilitaire de base en ajouter un !. Mais du coup, si on veut surcharger un utilitaire qui surchargeait déjà, et bien on se retrouve dans le cas initial, on est coincés !

→

Une bibliothèque javascript existe pour palier ce problème, elle s'appelle tailwind-merge. Elle fournit une fonction javascript qui s'assure que les utilitaires passés en second arguments seront appliqués quelque soit les utilitaires présents précédements.

Elle est capable de gérer les raccourcis comme x et y. Si vous l'utilisez dans tout votre projet, cela vous assure que vos utilitaires seront toujours pris en compte. En revanche cela ajoute une dépendance et une petite couche de calcul à vos composants.
